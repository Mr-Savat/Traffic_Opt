⚡ QUBO Pipeline — Every 30 seconds
Step 1
Read Real-Time Queue via TraCI
Count stopped vehicles (speed < 0.1 m/s) per edge:
moto_NS, car_NS, tuk_NS, moto_EW, car_EW, tuk_EW
Step 2
Compute Weighted Queue (Eq. 1)
L_NS = 0.60×moto_NS + 0.30×car_NS + 0.10×tuk_NS
L_EW = 0.60×moto_EW + 0.30×car_EW + 0.10×tuk_EW
Step 3
Build QUBO Matrix (Eq. 2)
H(x) = −(L_NS − L_EW) × x₀
Encoded as Binary Quadratic Model (BQM)
Step 4
Solve via Simulated Annealing
SimulatedAnnealingSampler (50 reads)
→ Finds x₀ that minimizes H(x)
→ x₀ = 1 : NS is more congested → NS Green
→ x₀ = 0 : EW is more congested → EW Green
Step 5
Apply Decision to SUMO
TraCI → setPhase("J0", 0) → NS Green
TraCI → setPhase("J0", 2) → EW Green
↩ t = t + 1  →  back to Simulation Step t  (until t = 3600)
📊 Post-Simulation — After t = 3600
Step 6
Export Results to CSV
